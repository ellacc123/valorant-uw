## About

Valorant @ UW is a Registered Student Organization at the University of Washington. This website serves the 300+ members of club, by acting as an online hub, connecting members, sharing events, and growing the UW Valorant community.

---

## Getting Started

```bash
# Clone the repo
git clone https://github.com/webimpactuw/valorant-uw.git
cd valorant-uw

# Install dependencies
npm install

# Start development server
npm start
```

Then open [http://localhost:3000](http://localhost:3000) to view it in your browser. The page will hot-reload as you make edits.
